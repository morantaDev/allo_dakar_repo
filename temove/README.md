# Allo Dakar

Application de réservation de taxi pour Dakar, développée avec Flutter.

## Structure du projet

- `lib/screens/` - Écrans de l'application
- `lib/widgets/` - Widgets réutilisables
- `lib/theme/` - Configuration du thème et couleurs
- `lib/utils/` - Utilitaires et helpers
- `lib/models/` - Modèles de données

## Fonctionnalités

- ✅ Écran d'accueil/Welcome
- ✅ Inscription/Connexion
- ✅ Géolocalisation et carte
- ✅ Réservation de course
- ✅ Suivi de course en temps réel
- ✅ Historique des courses

## Installation

```bash
flutter pub get
flutter run
```

## Configuration

Pour utiliser Google Maps, vous devrez configurer une clé API dans `android/app/src/main/AndroidManifest.xml` et `ios/Runner/AppDelegate.swift`.
